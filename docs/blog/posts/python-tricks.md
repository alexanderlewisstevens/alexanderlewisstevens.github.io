# Top 5 Python Tricks

1. List comprehensions: `[x*x for x in range(10)]`
2. Unpacking: `a, b = b, a`
3. F-strings: `f"Hello, {name}!"`
4. The `enumerate()` function for loops
5. Using `zip()` to iterate over multiple lists
